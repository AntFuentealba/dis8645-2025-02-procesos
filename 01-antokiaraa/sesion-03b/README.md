## repaso

pantalla OLED 128x64 y mini protoboard

tinkercad / wokwi

4 cables:

- **GND** cable principal de tierra, ground (*GND*)
- **VCC** voltaje de alimentación (importante saber el voltaje) (*5V*)
- **SCL** signal clock, tiene que saber "cuándo" (*A5*)
- **SDA** datos (*A4*)
